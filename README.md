# Pitch Page System

Professional ClickBank-ready pitch page for selling your cryptocurrency exchange script (or any other product).

## Features

### Public Pitch Page
- Logo (uploadable from admin)
- Big hero title + subtitle
- Screenshot carousel with arrows + lightbox for full-size viewing
- Red "Demo" button + Green "Buy Now" button with surrounding CTA text
- Post-purchase info card
- "6 months free support" highlighted callout
- Short description with link to full description page
- Top navigation: Home, FAQ, Refund, Privacy, Terms, Support (external link)
- Mobile-responsive footer with quick links and legal pages

### Admin Panel
- Secure login with bcrypt password hashing
- Dashboard with setup checklist
- 7-tab settings panel: Branding, Hero, CTA & Buttons, Info Text, Description, Footer, SEO
- Logo upload with preview
- Screenshots manager (multi-upload, captions, sort order, hide/show, delete)
- FAQ manager (categories, sort, status, HTML answers)
- Pages editor (Refund, Privacy, Terms, Full Description — all editable)
- Profile management with password change

### Pre-Seeded Content
- **30 professional FAQs** covering general questions, installation, features, pricing, support, customization, and hosting
- **Privacy Policy** — GDPR/CCPA-compliant with detailed sections
- **Terms of Use** — License terms, disclaimers, regulatory compliance, limitation of liability
- **Refund Policy** — 30-day money-back guarantee with ClickBank policy reference
- **Full Product Description** — Comprehensive feature list of the exchange script

## Installation

### 1. Upload
Upload `pitch_complete.zip` to your `public_html/` folder via cPanel File Manager or FTP.
Right-click → Extract. Make sure files are extracted directly into `public_html/`, not a subfolder.

### 2. Set Permissions
Set folder permissions to **755** (cPanel File Manager → Right-click → Permissions):
- `config/`
- `logs/`
- `uploads/`
- `uploads/logo/`
- `uploads/screenshots/`

### 3. Create Database
In cPanel → MySQL Databases:
- Create a new database (note the name, e.g., `youraccount_pitch`)
- Create a database user with a strong password
- Add the user to the database with **ALL PRIVILEGES**

### 4. Run Installer
Visit `https://yourdomain.com/install/` in your browser.

The 4-step wizard will guide you:
1. **Server requirements check** — verifies PHP version, extensions, writable folders
2. **Database connection** — enter the credentials from step 3
3. **Site & admin account** — set site name, URL, and create your admin login
4. **Done** — installation complete

### 5. Delete Install Folder
**IMPORTANT:** After the wizard finishes, delete the `/install/` folder from your server.

### 6. Configure
Log into the admin panel at `/admin/login.php` and:
1. Upload your logo (Settings → Branding)
2. Update the hero title/subtitle (Settings → Hero)
3. Set Demo button URL (your live demo) and Buy Now button URL (your ClickBank hoplink)
4. Set Support URL (where users go for help)
5. Upload screenshots
6. Customize FAQ entries
7. Review and edit Privacy/Terms/Refund pages with your business name and contact info

## Requirements

- PHP 7.4+ (8.0+ recommended)
- MySQL 5.7+ or MariaDB 10.3+
- PHP extensions: PDO, PDO MySQL, Mbstring, OpenSSL, JSON, GD, Fileinfo
- Apache with mod_rewrite (or Nginx with equivalent rewrite rules)

## Security

- All forms protected with CSRF tokens
- Passwords hashed with bcrypt (PASSWORD_DEFAULT)
- All database queries use prepared statements
- `config/`, `logs/`, `includes/`, `public/` folders blocked from direct browser access via .htaccess
- Upload folder blocks PHP execution
- Security headers: X-Content-Type-Options, X-Frame-Options, X-XSS-Protection, Referrer-Policy
- Admin lock file prevents accidental re-installation

## Customization

### Edit CSS
- Public site styling: `assets/css/public.css`
- Admin panel styling: `assets/css/admin.css`

CSS variables at the top of each file make color/spacing changes easy.

### Add New Pages
From admin: **Pages → New Page**
- Set slug (URL: `/page/{slug}`)
- Add HTML content
- Optionally show in footer

### Add New Settings
Edit the schema directly or add via SQL:
```sql
INSERT INTO px_settings (setting_key, setting_value, setting_group) 
VALUES ('your_key', 'your_value', 'your_group');
```
Then access in PHP with `settings('your_key')`.
